README to make API Calls

NY Times

Home and Section

In order to get home results and section results, need to give the following format:
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/home
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/world
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/politics
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/business
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/technology
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/sports

Search

In order to make search queries, need to give the following format:
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/search/:query

Detailed View

In order to make detailed article queries, need to give the following format
https://ivillega-nytimes-guardian.wl.r.appspot.com/nytimes/article_view/:articleID
